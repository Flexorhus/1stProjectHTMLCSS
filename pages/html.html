L'HyperText Markup Language HTML

Le HyperText Markup Language, abrégé HTML, actuellement sous la version HTML5, est le langage de balisage conçu pour structurer une page web. Ce langage permet d’écrire de l’hypertexte (d’où son nom), de structurer sémantiquement une page web, de mettre en forme du contenu, de créer des formulaires de saisie ou encore d’inclure des ressources multimédias dont des images, des vidéos, et des programmes informatiques.

Structure type d'un site HTML

Ci-joint un exemple type d'une page HTML avec :

  • Le !DOCTYPE qui stipule le type de document.
  • La balise html <html>.
  • La balise head <head> qui entoure la balise meta <meta> et la balise title <title>.
  • La <meta> permet de définir l'encodage du site à savoir la prise en charge des caractères spéciaux.
  • La <title> permet de définir le titre du site visible dans l'onglet du navigateur.
  • La <body> est le corps d'un site web.
  • La <nav> permet d'insérer une barre de navigation qui contient des sous-menus.
  • La <Ol> et <Li> permet d'insérer une liste non ordonnée
