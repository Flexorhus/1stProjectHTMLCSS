Le Cascading Style Sheets CSS

Le Cascading Style Sheets, abrégé CSS, actuellement sous la version HTML5, est le langage de feuille de style utilisé pour styliser une page HTML. Ce langage CSS décrit la façon dont les éléments doivent être affichés à l'écran, sur papier, à l'oral ou sur d'autres médias. CSS est l'un des langages principaux du Web ouvert et a été standardisé par le W3C, site qui permet de valider son code HTML ou d'autre contenu tel que le css.
