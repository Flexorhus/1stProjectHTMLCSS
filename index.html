L'UNIVERS de l'informatique est vaste et ne cesse de croitre avec le temps. Le brouillard ne fait que s'accentué avec les langages de programmation, mais par où commencer lorsque l'on souhaite devenir développeur. Nous allons tenter de démystifier tout cela.

Qu'est ce qu'un développeur ?

Le développeur web et web mobile s’occupe du développement informatique des sites web et web mobile, d'applications web et mobile, plateformes web.
Il développe de nouvelles fonctionnalités et/ou maintient les fonctionnalités existantes.
En plus de coder, un développeur peut intervenir dans la phase de conception, participer à l’élaboration d’algorithmes, prendre du recul sur les attentes de l’utilisateur final, rédiger de la documentation ou encore conduire des tests.

Selon l’étendue et les sujets sur lesquels il travaille, on parle notamment de :

  • Développeur Frontend
  • Développeur Backend
  • Développeur Fullstack

Grosso modo, Le développeur FrontEnd est en charge d’optimiser l’ergonomie et la navigation d’un site web. À Contrario le développeur BackEnd,a pour mission de concevoir tous les elements technique indispensable au fonctionnement d'un site web. Le développeur FullStack a pour principale mission la programmation d'une application ou d'un site, le concevoir de A à Z (création, développement, codage, maintenance ...).
Le développeur FullStack a des compétences dans tous les domaines : FrontEnd, BackEnd, UX/UI, Architecture.
